关于WhLUG


        WHLUG，是英文“Wuhan Linux User Group”的缩写，中文名称为“武汉Linux用户组”。

        WHLUG发起者为Linux Deepin社区和华中科技大学联创团队。

        WHLUG的目的在于活跃华中开源社区，推广开源软件，传播开源理念。

        我们认为WHLUG自创立之日起应拥有和坚持以下原则：
  •         独立自由。我们不受任何商业组织的控制，始终保持非营
  •         利性质。
  •         透明开放。我们对任何人或组织开放，我们的一切进行公
  •         示，包括财政、管理。
  •         社区民主。我们尊重每个成员的意见，由成员选举产生的
  •         主席团执行社区决策和负责社区日常管理。
我们坚持“独立自由”，但乐意与商业组织合作、接受赞助。
我们应该保持用户组的可持续发展。
我们欢迎更多的人们加入。

WHLUG.ORG域名由tuhaihe持有并进行志愿维护。
WHLUG.ORG网站由华中科技大学联创团队开发与维护。
WHLUG邮件列表：wuhanlug@googlegroups.com，可发送邮件到 wuhanlug+subscribe@googlegroups.com 进行订阅。


过往风云

        作为后来人去写前辈的历史，这是件谨慎的事情。岁月变迁，过往不曾留下过于浓重的痕迹，我们只能通过可查的资料来慢慢梳理脉络。

        我们检索到：

        “武汉GNU/Linux用户协会的前身是1999年2月成立的武汉Linuxer联盟，主要目的是想为Linux爱好者提供一个互相学习和交流的较为紧密的社区环境。

        这一后起之秀很快以更新快、内容多而在Linux爱好者中享有盛名。此后的成绩更引人注目，它先后和两家国内知名杂志合作推出了自由软件特刊，充分发挥了网络和传统媒体的优势，为网站的功用探索出一条新的路子。武汉自由软件协会的运作模式很有值得借鉴的地方。”（搜狐IT频道1999年文章[1]《Linux在中国之Linux User Group》摘录）

        “2、武汉自由软件协会（http://www.clinux.org）：武汉自由软件协会是一个非营利性学术团体，由自由软件及相关信息技术的爱好者、研究者、开发者组成，宗旨是发展自由软件的标准、技术与精神，促进商业合作与共识。它成立于1998年6月23日，是国内历史最悠久的自由软件社区组织之一。武汉自由软件协会经历了三个发展阶段：第一个阶段是1998年－2001年，主要致力于自由软件的推广宣传；第二个阶段是2002年－2004年，属于开源协作开发平台的建设阶段；第三个阶段是从2005年起至今，主要致力于开源产业相关企业的孵化。武汉自由软件协会还致力于开源协作开发平台的建设，建有GRO.clinux.org，是国内最早建设的也是规模最大的开源软件协作开发平台之一，目前有近千个个开源项目，近2万名注册开发用户。”[2]

        当时武汉GNU/Linux用户协会开启了一个Linux多线程下载软件项目，内部称为“芝麻开发计划”，由天意、深入主导。他们与《软件杂志》、楚天热线、武汉热线等杂志或网站保持积极的联系，他们承办了当时自由软件爱好者发起的《九章》杂志的部分期刊线上出版工作。

        他们在网络不发达的年代积极维护用户协会网站，他们积极原创教程、发表自己对业界的看法与观点。他们经常通过IRC进行线上沟通与交流，他们也经常在大学等地点进行周末聚会。他们积极联络全国其他Linux用户协会并与之开展交流，经常讨论自由软件对中国IT产业的影响，在雅虎邮件列表中可以经常看到“天下兴亡，我的责任”的签名邮件。
